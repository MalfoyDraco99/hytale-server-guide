The (Un)-Official Hytale Server Guide

This guide is made for Windows users, but should probably work for Linux.

Requirements:

A Decent PC (At minimum 16GB Of RAM + a goodish CPU)

Elevated Privileges (Having administrator on your pc, if your the only account on your pc. Dont worry about this)

The admin password for your router, if you don't know how to find it, you will find out later in this guide.

Step 1: Downloading

Download the Hytale server downloader: Click here to download or, if you don’t trust me, google "Hytale Server Manual" and find the download there.

Step 2: Unzipping

Open your downloads folder and unzip the downloaded folder.

Step 3: Running the EXE

In the unzipped folder, there should be a file named hytale-downloader-windows-amd64.exe. Double-click it to run.

Step 4: Authenticating

At this point, it will ask you to authenticate your Hytale account. Click the link it sends and enter the authentication code.

After you do this, it should download a zip, unzip it when the download is done. Inside it should be Assets.zip and a Server folder. Open the Server folder.

Step 5: Downloading Java

Download Java 25 by Adoptium: Here

After going to that link, press Java 25 - LTS, And click the download button under "Windows"

Step 6: Installing Java

Run the .msi installer and follow the instructions that it says, if there is an option called "Add JAVA to PATH or Set JAVA_HOME variable" make sure it is checked / enabled.

Step 7: Enabling File Extensions

Follow the images below, if you are not on windows 11 google how to do it. You're a big boy. I believe in you.

Step 8: Creating start.bat

Create the start.bat folder to start your server. Right click the empty space inside the Server folder Hover over "New" then press "New Text Document"

Step 8.5: Creating start.bat

Click the text document you created once, then press F2 on your keyboard, rename it to start.bat

Step 9: Writing start.bat

Right click start.bat, hover over "Open With" and press "Notepad"

Insert the batch code below into the space you see, once done press CTRL + S on your keyboard

@echo off
java -jar HytaleServer.jar --assets ../Assets.zip
pause

Step 10: Running start.bat

Double tap start.bat to run it, it should start up. If it opens notepad instead of starting up, visit the FAQ at the bottom of this page. If there are warnings, ignore them. If it displays this message at the end it was successful! Congrats!

Step 13: Port Forwarding - Router Gateway

Here's when things get fun, and by fun, I mean semi complicated.

Type "cmd" into your windows search bar, then press enter, then type "ipconfig", then find the numbers highlighted in the image. The one that it is under will depend on what connection you are using.

Step 13.5: Port Forwarding - Entering your ISP's Network Page

Take the numbers under default gateway, and type them into your search bar on your search engine of choice, like your typing a link.

Step 13.6: Port Forwarding - The Real stuff

On the webpage, enter your admin credentials to your router. Usually found on a sticker on the back of your router, and then port forward port 5520. If you are confused visit Here

FAQ / Support:

Q: How do I stop my server?
A: Stop your server by running the stop command in your console. This is the healthiest way to do it that minimizes the chances of dataloss.

Q: My ISP doesn't let me port forward, can i still do this?
A: I hate to be the bearer of bad news here, but no you can't do this. It simply wont work.

Q: My server.bat opens in notepad instead of as a command terminal!
A: Type "cmd" in your file address bar (highlighted in the image) while having your servers folder open. Then type start.bat
